UASerials — Lampa Plugin (GitHub Pages)

Цей репозиторій містить файл online_mod.js для підключення до Lampa як онлайн-мод.

  1. Увімкни GitHub Pages для цього репозиторію (Branch: main, Folder: /root).
  2. Скопіюй посилання на online_mod.js:
    https://<твій-юзер>.github.io/<твій-репо>/online_mod.js
  3. Додай це посилання у Lampa → Налаштування → Онлайн-моди.

За замовчуванням мод звертається до адаптера https://lampa-parser-lime.vercel.app. Можна змінити у коді або через window.UAS_ADAPTER.
